NEEL-IT @ EVALITA 2016
======================

This package contains the development data for the EVALITA-2016 NEEL-IT task.
The file containing the orginal text of the tweets will be released to registered participants only.


1) README: this file

2) dev_id_neel_it16_v1.tsv: a file which contains the tweet ids.

3) dev_gold_neel_it16_nil_v1.tsv: a tab separated value file which contains fields: the tweet id, the start offset of the annotation starting from 0, the end offset of the annotation, link to canonical version of DBpedia, entity type. NIL istances have been clustered, all the mentions of the same entity will appear with the same NIL identifier.
